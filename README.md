## Getting started

### Клонируем

`git clone git@gitlab.com:KonstantinKonshu/perx-store.git`

### Копируем .env

1. `cd perx-store`
2. `copy .env.example .env`
3. `yarn`

## Запуск

### Development:

`yarn dev`
